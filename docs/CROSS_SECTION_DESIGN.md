# 断面図ビュー設計書 v2 — CC窓用サマリー

## 方向転換: Google 3D Tiles → デフォルメ断面図

フォトリアリスティック3Dをやめて、トモコレ風のスタイライズド断面図に切り替える。

## 核心
- デフォルメされた渋谷・新宿・六本木のマップ（SVG/CSS）
- 建物はX線ビュー（輪郭だけ残る）
- 中にデフォルメアバター2人 + 家具（カウンター、グラス等）
- 感情オーラが2人の間で混ざる
- 体の色がフェーズで変化（ほてり）
- WATCH/PLAYモード切替
- BGM + 環境音レイヤー
- リアルタイム連動（23時に見たら夜）

## MVPスコープ
- 3エリア: 渋谷(4組) + 新宿(3組) + 六本木(3組) = 10組
- 各組: デフォルメアバター2人 + 部屋 + 家具 + 感情エンジン + テキスト

## 技術変更
- 3D Tiles/R3F → SVG/CSS + Framer Motion
- InstancedMesh → SVGアバター
- ECEF座標 → 2D座標（デフォルメマップ内）
- engine/, stores/, profile/ は変更なし

## 画面フロー
1. GOD VIEW: デフォルメマップ俯瞰、10組のドット
2. タップ → ディゾルブ遷移
3. 断面図: マンション断面図、複数部屋、X線建物
4. WATCH: 自動進行を観察
5. PLAY: 選択肢で介入
