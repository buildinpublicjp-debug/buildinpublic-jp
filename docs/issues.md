## 🔴 CRITICAL（静止画を生きてるように見せる）

- [ ] #049 背景画像の微呼吸 — CSSでscale(1.0)→scale(1.01)を6秒周期でゆっくり往復（transform + ease-in-out infinite）。人が呼吸してるように画像全体がわずかに膨張・収縮する。肉眼でギリギリ気づくレベルの微細さ
- [ ] #050 暖色ライトのゆらぎ — 画像の上にオレンジ半透明オーバーレイ(rgba(255,160,60,0.03))を重ねて、opacityを0.02〜0.05で3秒周期で揺らす。バーの照明が微妙にゆらいでるように見える
- [ ] #051 ボケ光パーティクル — 画面上部に5〜8個の小さい円(8-15px)を半透明白(rgba(255,255,255,0.05-0.15))で浮かべて、ゆっくり上に浮遊(translateY -20px, 8秒周期)。ピントが合ってない光の粒。position:absolute、pointer-events:none
- [ ] #052 フィルムグレイン — 画面全体にノイズテクスチャのオーバーレイ。CSS background-image: url(data:image/svg+xml の小さいランダムドットパターン)をrepeat、opacityを0.03〜0.06で微妙に変化。映画フィルムのザラつき感

## 🟡 IMPORTANT（UIを生き生きさせる）

- [ ] #053 感情バーの脈動 — 各バーの先端にbox-shadow(0 0 6px color)をつけて、opacityを0.5〜1.0で1.5秒周期でpulse。心拍のように脈打つ
- [ ] #054 「接近」テキストのアニメーション — 右上の「接近」「残り15.1h」の文字をopacity 0.6→1.0で2秒周期でゆっくり点滅。時間が動いてる感じ
- [ ] #055 カップル切替時のシーン遷移 — ◀ ▶ でカップル変更時に、画像をcrossfade（opacity: 1→0→1、0.4秒）。パッと切り替わるんじゃなくて溶ける
- [ ] #056 テキストの出現アニメ — カップル切替時に上部の名前・MBTIがfadeIn+translateY(10px→0)で0.3秒かけて出現。下部の分析パネルも0.1秒遅れてfadeIn
